\doxysubsection{Detailed Description}
Local Queue for Jobs. This class is responsible for managing the local queue of jobs that will be processed by each worker thread. 

Holds the Jobs that need to be processed by each worker thread. The \doxylink{class_local_job_queue}{Local\+Job\+Queue} class provides methods to add jobs to the local queue and process the jobs in the local queue.
