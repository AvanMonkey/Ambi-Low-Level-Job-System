\doxysubsection{Detailed Description}
Holds the number of threads available on the device\textquotesingle{}s thread pool. 

Initialises a vector containing the amount of worker threads (-\/1 for the main thread) available on the device. This can then be used to assign these threads jobs and use them for other functions
